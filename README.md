# 📊 Retail Sales Forecasting & Inventory Optimization System

🚀 An end-to-end, industry-oriented analytics system designed to forecast demand and optimize inventory decisions in retail businesses.

---

## 💡 Problem Statement

Retail businesses often struggle with:

- ❌ Overstock → Dead inventory & capital loss  
- ❌ Stockouts → Lost sales & unhappy customers  

👉 The key challenge:  
**“How much inventory should we actually keep?”**

---

## 🎯 Solution

This project builds a **complete data-driven system** that:

✔ Forecasts future demand using Machine Learning  
✔ Calculates optimal inventory levels  
✔ Identifies low-stock risks  
✔ Provides actionable business insights  

---

## 🧠 Key Features

- 📈 **Demand Forecasting Engine** (Random Forest Model)  
- 📦 **Inventory Optimization Logic** (Safety Stock & Reorder Point)  
- 📊 **KPI Dashboard** (Revenue, Demand, Alerts)  
- 📉 **Sales Trend Analysis**  
- 🏪 **Multi-Store & Multi-Category Simulation**  
- ⚠️ **Critical Stock Alerts**  
- 🎯 **Interactive Streamlit Dashboard**
